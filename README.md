1. Kotlin
2. Android Studio
3. Firebase

  b. Realtime Database
  c. Firebase Cloud Messaging
  d. Firebase Storage
4. Sqlite
5. Recycler View
6. Button Navigation

## Fitur/teknologi yang ada diaplikasi
1. Alaram Manager
2. Chat
3. Fitur Pencarian
4. Notifikasi
